Introduction

The simplest model and the foundation of many other models is the SIR model.

  • S: Suseptible
  • I: Infectious
  • R: Recovered

Note that R can be either people that have recovered from the disease or also people that have died. This model does not differentiate the two states but later on we can look at how we can extend this model using the SIRm model where m incorporates the people that have passed away.

You can think of these models as a network model where each node consists of the compartments and the edges from each node corresponds to a rate in which activity from one node to other occurs. These variables (S, I, and R) represent the number of people in each compartment at a particular time. To represent that the number of susceptible, infected and recovered individuals may vary over time (even if the total population size remains constant), we make the precise numbers a function of t (time): S(t), I(t) and R(t). $\beta$ is the rate at which S becomes I and $\gamma$ is the rate at which I becomes R.

graph LR S-->|beta|I I-->|gamma|R ODE --> Parameter Parameter --> Model Model --> Prediction

As implied by the variable function of t, the model is dynamic in that the numbers in each compartment may fluctuate over time. During an epidemic, the number of susceptible individuals falls rapidly as more of them are infected and thus enter the infectious and recovered compartments. The disease cannot break out again until the number of susceptibles has built back up, e.g. as a result of offspring being born into the susceptible compartment.
